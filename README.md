# Expense Tracker Dashboard

## Project Overview

The Expense Tracker Dashboard is a web-based financial management application developed using HTML, CSS, and JavaScript. The application helps users record, monitor, and analyze their daily income and expenses through an interactive dashboard interface.

The system provides an easy way to manage personal finances, track spending habits, maintain transaction records, and improve budgeting decisions.

---

## Objectives

The primary objectives of this project are:

* Track daily expenses and income.
* Manage personal financial records.
* Monitor spending habits.
* Calculate available balance.
* Improve budgeting and financial planning.
* Provide financial insights through a dashboard.

## Application Workflow

### Step 1: Open Dashboard

Users access the Expense Tracker Dashboard through index.html.

### Step 2: Add Income

Users enter income details and save financial records.

### Step 3: Add Expenses

Users record expenses by entering:

* Expense Name
* Category
* Amount
* Date

### Step 4: Automatic Processing

JavaScript processes transactions and updates financial data.

### Step 5: Dashboard Update

The system automatically updates:

* Total Income
* Total Expenses
* Available Balance

### Step 6: Financial Monitoring

Users monitor spending patterns and financial performance through the dashboard.

## How to Run the Project

### Using VS Code

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open index.html.
4. Right-click and select "Open with Live Server".
5. The Expense Tracker Dashboard will open in your browser.

### Using a Browser

1. Open the project folder.
2. Double-click index.html.
3. The application will launch automatically.
